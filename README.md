# PokeDex
an exploration of React Router

## Project Overview:
*The purpose of this project is to showcase the capabilites of React Router. Namely:*
 - Nested Routing
 - Dynamic Routing
 - Declarative Routing
 - URL Manipulation
 - Route Matching
 - Code Splitting

## To Install:
Fork and clone, then:
```
npm install
npm start
```
